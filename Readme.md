### How To Run

-  To Install Dependencies:
```
npm i
```

- EX. of Transaction:
```
Leeus.createTransaction(new Transaction(Date.now(), "wallet-a", "wallet-b", 10));
```

- EX. of Mining:
```
Leeus.mineCurrentBlock('wallet-k');
```
